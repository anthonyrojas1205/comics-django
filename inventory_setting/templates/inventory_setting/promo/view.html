Product Tags

Tags help you to manage discounts on your products. You can create a tag, set a discount for that tag and then apply the tag to products to discount them. Tags will show up on your store's front page as well, so if you are having a sale for instance and create a tag called 'Summer Sale', any products with the 'Summer Sale' tag will be shown on the front page, and discounted with the 'Summer Sale' tag's discount.
